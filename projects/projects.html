What have I been up to?

Finance Tracking App

Being quite a saver I decided to learn Android Development by making my own app. The app had a single simple purpouse - to track my expenses as months fly and help me save money.

In the begining app was qute simple, but as time went by it got more complex. At present time it is using Java as programming language, as well as Android and Spring frameworks.

If you want to check the source? Just go to my GITHUB page.

So, what can you do with it?

  • You can add remove expenses as you wish.
  • If you are too lazy to type you can simply scan Receipt QR (only for Bulgarian recepts)
  • Each expense has a price, category, date and name.
  • Each expense has an optional list of items that it included - kind of like a receipt.

Ok, adding expenses. What else?

  • Monitor key data on your Dashboard.
  • Customize Dashboard to fit your needs.
  • Analyze data using build-in graphics.
  • Browse and filter expenses.
  • You can export expenses as csv file.
  • Automatic syncing to a remote server.

And what did I learn from it?

Imporved my Java, Android & Spring frameworks knowledge.

Tried to follow OOP and some of the simpler Design Patterns

Understood basic ORM via Room and Hibernate as well as how to structure simple Databases.

Gained basic understanding of Gradle.

Made use of Git to share my project on Github.

Encountered firsthand problems of project-scaling and constant change
